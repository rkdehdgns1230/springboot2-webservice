## 다음과 같은 기술 및 주제를 다룰 수 있었습니다.
- Spring Boot
- Spring Data JPA
- Unit Test 작성 방식 + TDD와 Unit Test의 개념에 대해서
- Mustache를 이용한 view 생성
- Spring Security
- AWS EC2, RDS, CodeDeploy, S3 + IAM 등..
- Travis CI를 이용한 CI 단계 구축
- 프로젝트 빌드 배포 과정 및 이를 자동화 하는 경험
- nginx를 이용한 무중단 배포
